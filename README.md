# Arduino_ile_Kizil-tesi_Kumandal-_Arac
Arduino kullanılarak geliştirilen kızılötesi (IR) kumanda kontrollü bir robot araç sistemidir. Araç; ileri, geri, sağ ve sol hareket edebilmekte, dar alanlarda manevra yapabilmektedir, ayrıca dans modları vardır. Proje, motor sürücü ve IR alıcı modül kullanılarak uzaktan kontrol edilecek şekilde tasarlanmıştır.
